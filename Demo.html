Link Prediction in Social Networks using Machine Learning

Recommending "Streamers to Follow" on Twitch

Twitch is a video live streaming service operated by Twitch Interactive, a subsidiary of Amazon

Research Question : Link Prediction in Social Networks -¶

Given a social network graph, link prediction is the problem of predicting which new links could be formed some time in the future. This is useful in various tasks such as recommending new friends on Facebook or new topics on Twitter.

Base Paper -¶

The Link-Prediction Problem for Social Networks
LibenNowell, D. and Kleinberg, J. (2007), The linkprediction problem for social networks. J. Am. Soc. Inf. Sci., 58: 1019-1031. doi:10.1002/asi.20591

def plot_roc(y_test, y_pred):
    fpr,tpr,ths = roc_curve(y_test, y_pred)
    auc_ = auc(fpr, tpr)
    plt.plot(fpr, tpr, color='tab:blue',label='ROC curve (area = %0.2f)' % auc_)
    plt.xlabel('FPR')
    plt.ylabel('TPR')
    plt.title('ROC Plot')
    plt.legend()
    plt.show()

def plot_confusion(y_test, y_pred):
    
    C = confusion_matrix(y_test, y_pred)
    
    A =(((C.T)/(C.sum(axis=1))).T)
    B =(C/C.sum(axis=0))
    plt.figure(figsize=(3,3))
    
    labels = [0,1]
    cmap=sns.light_palette("tab:blue")
    sns.heatmap(C, annot=True, cmap=cmap, fmt=".3f", xticklabels=labels, yticklabels=labels)
    plt.xlabel('Predicted Class')
    plt.ylabel('Original Class')
    plt.title("Confusion matrix")
    plt.show()
